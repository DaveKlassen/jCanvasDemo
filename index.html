JCanvas Examples

I chose to use the JCanvas plugin for JQuery for this assignment, which can be downloaded from Caleb Evans' personal site. According to the author he has been developing websites since 2010. JCanvas is a way for us to make use of the Canvas via. JQuery. As most of the examples show JCanvas has been used for quite a few 2D projects, but I could not find any 3D examples to learn from (At least not along the lines of Banana Bread).

For working with the plugin I chose to review the examples provided by the Mozilla Canvas Tutorial and gain an understanding for what I could do with the JCanvas interface, mostly because after reading the documentation website I didn't see anything that excited me, and I did want to learn much more about what I could do with Javascript and a HTML5 Canvas Object.
